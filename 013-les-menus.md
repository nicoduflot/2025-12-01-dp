# Les menus

* Les menus sont gérés dans :
Accueil > Administration > Structure > Menus
* Pour la navigation principale, il faut chercher la navigation dans la liste et cliquer sur "Modifier le menu"

Pour créer un lien, on clique sur "Ajouter un lien"
Il faut remplir le texte du lien et la destination
* Si le lien ne sert qu'a contenir des lien de sous-menu, dans la destination il faut écrire ```<nolink>```
* C'est dans la page du menu qu'il est possible de réordonner les liens, et de décaler vers la gauche sous un lien parent pour créer des lien de sous-menu
