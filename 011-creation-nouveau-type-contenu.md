# Création des nouveaux types de contenu

## méthode de création d'un contenu

* Accueil > Administration > Structure > Types de contenu > Ajouter un type de contenu
* Créer le type de contenu
* Créer le pattern pathauto
* Gérer les champs
    * Il est possible, voir recommandé, de réutiliser le maximum de champs créés pour d'autre contenu, pour simplifier la maintenanance et surtout grand intérêt lors de la création des vues
    * créer les champs nécessaires
* Gérer l'affichage du formulaire
* Gérer l'affichage (vue visiteur)
* Gérer les droit : qui à les droits de ajout, modification, suppression, etc.

## Créateur / Créatrice

### Les champs

* Nom ( il s'agit du champ titre déjà présent de base pour tout nouveau type de contenu, renommé)
* Portrait (champ média image limité à 1)
* Biographie (il s'agit du champ Corps réutilisation du champ corps utilisé dans page et article)

## Lire

### Les champs

* Titre (déjà présent de base pour tout nouveau type de contenu)
* Résumé (il s'agit du champ Corps réutilisation du champ corps utilisé dans page et article)
* Taxonomie support
* Taxonomie genre
* Galerie de Media (couverture de l'oeuvre, images supplémentaires possible)
* Créateur / Créatrice (champ référence au contenu Créateur / créatrice, illimité car plusieurs peuvent intervenir sur un contenu)
