# Ajouter - configurer une taxonomie à un contenu existant

## Ajouter la taxonomie

* Accueil > Administration > Structure > Types de contenu > Article

on arrive sur la page "Modifier le type de contenu *Article*"

* Cliquer sur "gérer les champs"
* Cliquer sur "Créer un nouveau champ"
* Choisir le type du champ (ici Référence)
    * Une taxonomie est une référence à du contenu classé, et non pas un contenu particulier
* Cliquer sur "continuer"
* Indiquer le libellé du champ
* Cocher "Terme de taxonomie" dans le référence
* Cliquer sur "continuer"
* On configure le champ : 
    * Le libellé et le type de champ référence sont déjà remplis
    * Nombre de valeurs autorisée : 
        * on peu limiter le nombre de terme pour ce contenu, ou laisser la possibilité de choisir autant de terme que nécessaire pour classer ce contenu
    * Dans type de référence, "Créer les entités référencées si elles n'existent pas déjà" permet de laisser aux auteurs la possibilité de créer directement de nouveaux termes dans le vocabulaire si nécessaire
    * Bien indiquer que le vocabulaire référencé est "Catégories"
    * Il est possible d'indiquer une valeur par défaut dans ce champ à la création d'un contenu
* Enregistrer les paramètres.

## Gérer l'affichage de la taxonomie dans le formulaire de création / modification du contenu

Aller sur "Gérer l'affichge du formulaire"

Il est possible de :

* Changer l'ordre d'affichage des champs quand on ajoute / modifie le contenu
* Changer le mode de saisie des champs.
