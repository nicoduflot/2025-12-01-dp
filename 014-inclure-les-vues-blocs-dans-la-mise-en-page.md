# Inclure les blocs dans des régions

Quand on a créé une vue bloc, il est possible de l'inclure dans la structure du site grâce à la mise en page des blocs

* Accueil > Administration > Structure > Mise en page des blocs
* Les lien "Aperçu des régions des blocs" permet de voir où se trouvent les "régions" de blocs

## Région

Une région est un emplacement dans la structure de site dans laquelle des blocs existent et dans laquelle il est possible d'ajouter d'autres blocs

## Bloc

Il s'agit d'élément préconstruit HTML permettant d'afficher du contenu.

* Il existe des blocs pré-éxistants (ex. Navigation principale)
* Les vue sous forme de bloc (Ex. Derniers Articles)
* Il est aussi possible de créer ses propres blocs
